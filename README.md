# Homebrew formulae

A custom [Homebrew tap](https://github.com/Homebrew/brew/blob/master/docs/brew-tap.md).

## Usage

Tap this repository:

```sh
brew tap cogito-lang/formulae
```

Install the packages it contains like any other Homebrew package:

```sh
brew install cogito
```
