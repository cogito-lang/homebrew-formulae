# Homebrew Formulae

A [Homebrew tap](https://github.com/Homebrew/brew/blob/master/docs/brew-tap.md) for Localytics projects.

## Usage

Tap this repository:

```sh
brew tap localytics/formulae
```

Install the packages it contains like any other Homebrew package:

```sh
brew install cogito
```
